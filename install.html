Installation

B-Translator is made of two parts, a client application and a server application, which communicate with each-other through an HTTP API. They can be installed together on the same server, or they can be installed on different servers. Both of these applications are built with Drupal7. For more flexibility, there is also a drupal module for the client, that can be installed independently on any Drupal application or distribution.

Installing B-Translator Client

Assuming that the host system is Ubuntu, we can install in it B-Translator Client like this:

  1. Get and extract the B-Translator Client TGZ archive : 
    wget https://googledrive.com/host/0B7H_DCc2DjNGamstaDd5VjBydUk/bcl-20140917.tgz
sudo tar --extract --absolute-names --gunzip --file=bcl-20140917.tgz
    This will create the directory /var/chroot/bcl/ containing the chroot system, and the init script /etc/init.d/chroot-bcl.
  2. Customize the configuration of the chroot system: 
    sudo /etc/init.d/chroot-bcl start
sudo chroot /var/chroot/bcl/
cd /var/www/bcl/profiles/btr_client/
install/config.sh
  3. Fix also the network connection from inside chroot: 
    sudo chroot /var/chroot/bcl/
vim /etc/resolv.conf
ping www.google.com
  4. If this is a test installation or you are installing it on your local machine, add to /etc/hosts a line like this 127.0.0.1 example.org with the domain that you used in configuration and the IP of the server. Now you can open in browser https://example.org and access the application. You will have to confirm the security exception, since the SSL certificates of the web server are not real ones.

For other systems (Debian, Fedora, CentOS, etc.) small modifications to the init script /etc/init.d/chroot-bcl may be needed. For other ways of installation, which actually build the system from scratch, look at the file README.org of B-Translation Client.

Note: A version of the client based on Open Atrium is available as well in this archive bcl-oa-20140917.tgz. However it is not finished yet and needs heavy customization. Try it only if you are a Drupal developer and are confident in your skills.

Installing B-Translator Server

Installing a server is almost the same as installing a client, but we use the B-Translator Server TGZ archive instead. 

wget https://googledrive.com/host/0B7H_DCc2DjNGamstaDd5VjBydUk/btr-20140918.tgz
Also replace bcl with btr in the instructions above and add a domain for the server at /etc/hosts, for example 127.0.0.1 btr.example.org.

Look also at how to build the system from scratch at README.org.

Installing btrClient Module

The module btrClient is intended for Drupal developers who want to integrate it into their Drupal application. For instructions about it see https://github.com/B-Translator/btrClient or https://drupal.org/project/btrclient.

Vocabulary iFrame

Embed something like this in your web page or application:

<div style="width:300px; display:inline-block;
            float:right; border:1px solid #aaa; padding:5px;">
  <h3 style="margin-top:0px; color:#2a2;">Vocabulary: ICT_sq</h3>

  <iframe src="https://l10n.org.al/vocabulary/ICT_sq?display=iframe"
          width="100%" height="350px"
          seamless="seamless" frameborder="0">
  </iframe>
</div>
